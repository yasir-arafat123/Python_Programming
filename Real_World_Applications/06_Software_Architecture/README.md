## Learning Outcomes
- Identify and isolate domains and boundaries.
- Apply hexagonal/ports-adapters to decouple infrastructure.
- Implement plugin-based extensibility using entry points/importlib.
- Select appropriate pattern vs overengineering.

## Exercises
- Refactor a coupled script into layered architecture.
- Implement strategy vs if/elif chain.
- Add plugin discovery to extend a command set.
