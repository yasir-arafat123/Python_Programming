# CLI Tooling & Automation

Purpose: Build robust, user-friendly command line tools and automation scripts.

## Subtopics
1. 01_argparse_Basics
2. 02_click_vs_typer
3. 03_Command_Structure_subcommands
4. 04_Color_Rich_Progress

## Learning Outcomes
- Design ergonomic CLI argument structures.
- Compare argparse vs click/typer trade-offs.
- Add progress bars and color output.
- Package a CLI with entry points.

## Exercises
- Rebuild a script using typer with autocompletion.
- Add a progress bar for batch file processing.
- Create a task automation file (invoke) for common dev tasks.
