# Editable & Local Packages

## Key points

- `pip install -e .` for editable installs during development.
- Installing local directories and local archives (`pip install ./mypkg` or `pip install mypkg-0.1.tar.gz`).
- Using `pyproject.toml` + PEP 517/518 build backends vs legacy `setup.py`.
