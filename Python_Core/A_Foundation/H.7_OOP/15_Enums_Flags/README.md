# Enums & Flags
Purpose: Define a set of named constants to improve code readability and type safety.
## Key Concepts
- \Enum\ class basics
- Iterating over Enums
- Comparing Enums (identity vs equality)
- \IntEnum\ for integer compatibility
- \Flag\ and \IntFlag\ for bitwise operations
- \uto()\ for automatic value assignment
## Why use Enums?
- Prevent "magic numbers" or "magic strings"
- Group related constants
- Type safety (mypy understands Enums)
