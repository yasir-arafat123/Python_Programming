# Random Module
Purpose: Generate pseudo-random numbers, shuffle sequences, and make random selections.
## Key Concepts
- \andom.random()\ vs \andom.randint()\ vs \andom.uniform()\
- \andom.choice()\ and \andom.choices()\ (weighted)
- \andom.shuffle()\
- \andom.sample()\ (unique selection)
- Seeding with \andom.seed()\ for reproducibility
- Cryptographically secure randoms: \secrets\ module (vs \andom\)
## Examples
- Simulating a dice roll
- Shuffling a deck of cards
- Generating a random password (use \secrets\)
