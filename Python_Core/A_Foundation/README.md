# A_Foundation: Core Python Learning
**Purpose:** Master the fundamentals of Python programming - syntax, data structures, functions, OOP, file handling, and essential development practices.
**Target Audience:** Beginners to intermediate Python developers building a solid foundation.
## Module Overview
### A.0_Getting_Started
- Installation and setup
- First Python program
- IDE configuration
### B.1_Setup_Packaging
- pip and package management
- Virtual environments
- Module imports and packages
- Project structure (src layout, pyproject.toml)
### C.2_Core
- Python syntax fundamentals
- Variables, data types, operators
- Built-in functions
- Memory model and mutability
- Type system basics
### D.3_Strings
- String operations and methods
- Formatting (f-strings, format(), %)
- Regular expressions
- Unicode and encoding
- Text processing
### E.4_ControlFlow
- Conditionals (if/else, match)
- Loops (for, while)
- Exception handling (try/except)
- Control statements (break, continue, pass)
- Walrus operator
### F.5_Functions
- Function definition and arguments
- *args and **kwargs
- Lambda functions
- Scope and namespaces
- Decorators
- Context managers
- Closures and callbacks
- Partial functions
### G.6_Collections
- Lists, tuples, sets, dictionaries
- Comprehensions
- Iterators and generators
- collections module (defaultdict, Counter, deque, etc.)
- itertools, heapq, bisect
- Slicing and copying
### H.7_OOP
- Classes and objects
- Inheritance and polymorphism
- Encapsulation
- Dunder methods and dataclasses
- Composition vs inheritance
- Protocols and duck typing
- Metaclasses (basics)
- Method resolution order
- __slots__
### I.8_Files_Data
- File I/O (open, read, write)
- Paths (pathlib, os.path)
- JSON, CSV, XML, YAML
- Binary files and pickle
- Dates and datetime
- SQLite basics
- Config files
- Temporary files
### J.9_Quality
- Testing (pytest, unittest)
- Linting and formatting (ruff, black)
- Type checking (mypy, pyright)
- Code coverage
- CI/CD basics
- Pre-commit hooks
- Documentation standards
- Security scanning
### K.10_Advanced
- functools (lru_cache, partial, singledispatch)
- Descriptors (advanced attribute access)
## Learning Path
**Recommended Order:**
1. A.0 ??? B.1 ??? C.2 (Setup and basics)
2. D.3 ??? E.4 ??? F.5 (Core language features)
3. G.6 ??? H.7 (Data structures and OOP)
4. I.8 ??? J.9 (Files and quality practices)
5. K.10 (Advanced features when ready)
**Time Estimate:** 8-12 weeks for comprehensive mastery with daily practice.

After completing Foundation, proceed to **B_Advanced** for:
- Advanced typing and static analysis
- Robust error handling patterns
- Logging and observability
- Concurrency and parallelism
- Performance optimization
- Python internals
- Security best practices
- Deployment strategies
- Documentation systems
- Refactoring techniques
