# Archiving (zipfile, tarfile)
Purpose: Reading and writing archive formats.
## Key Concepts
- \zipfile\ module: Reading/writing .zip files
- \	arfile\ module: Reading/writing .tar, .tar.gz files
- \shutil.make_archive()\ (high-level shortcut)
## Common Tasks
- Compressing a directory
- Extracting all files
- Inspecting archive contents without extracting
