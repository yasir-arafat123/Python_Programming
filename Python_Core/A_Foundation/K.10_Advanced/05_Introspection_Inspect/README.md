# Introspection & Inspect Module
Purpose: Examining live objects, source code, and stack frames at runtime.
## Key Concepts
- Built-ins: \dir()\, \	ype()\, \id()\, \getattr()\, \hasattr()\
- \inspect\ module:
    - \inspect.getsource()\
    - \inspect.signature()\
    - \inspect.getmembers()\
    - \inspect.stack()\
## Use Cases
- Debugging tools
- Auto-documentation generators
- Frameworks that rely on magic (like pytest fixtures)
