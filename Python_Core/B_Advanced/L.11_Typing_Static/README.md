## Learning Outcomes
- Understand benefits and trade-offs of static typing in Python.
- Write clear function and variable annotations.
- Use generics, Protocol, TypedDict, and Literal effectively.
- Configure and interpret mypy/pyright reports.
- Integrate type checking into CI.

## Exercises
- Add typing to an untyped module and measure mypy warnings.
- Refactor a duck-typed API using Protocol.
- Create a TypedDict and migrate to a dataclass.
